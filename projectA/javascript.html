JavaScript

JavaScript is a client-side programming language that runs inside a lcient browser and proccess
commands on a computer rather than a server. Commonly used inside an HTML or ASP file. JavaScript
is not related to Java despite the name. JavaScript is a dynamically typed language basically meaning
the user can declare a vairable wihtout having to put anything into that variable. If the variable is
dynamic it can be anything- a bumber or a sentence.

What is JavaScript used for?

  • Adding interactive behavior to web pages
  • Creating web and mobile apps
  • Building web servers and developing server applications
  • Game development
