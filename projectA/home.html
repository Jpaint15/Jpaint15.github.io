Programming Languages

Programming languages allow us to give instructions to a computer in a way the computer will understand.
The language that a computer understands is called binary, and the process of translating programming language
to binary is known as compiling. There are many different programming languages, which each have their own
distinct features but all share common attributes. These languages allow computers to qucikly process information written
through code. We are going to talk about 4 different programming languages that will be discussed is C, Java, JavaScript, and Python.

Syntax

The syntax of a programming lanugage dsescribes which strings of characters that makeup a program. Syntax is
important so that the programmers can write valid programs. Wtihout syntax, the semantics of a code is nearly impossible to
understand. Every language has its own set of rules that make up its basic synax. Here are some of the examples or basic syntax:

  • Case Sensitive: Indentiers such as world and World have different meanings
  • Class Names: Some languages requre that firest letter of each word in class names be upper case
  • Program Filenames: The name of a program file must match the class name with the extension.

RAPTOR Flowchart Application

RAPTOR is a flowchart-based programming environment designed to help student visualize their algorithms without focusing
on the syntax of the program. RAPTOR programs are cretaed visually and exectuted visually by tracking the execution throught
a flowchart. Flowchart's are easier for the mind to undertand while looking at as opposed to looking at lines of code and
not being able to focus on the functionality of the program.
