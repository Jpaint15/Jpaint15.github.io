Java

Java is a general-purpose, object-oriented, high-level programming language with distinct features that make it ideal for software development. Java is used to develop applications for video games and mobile apps, as well as to create web applications. Programs that use or are written in Java include Adobe Creative Suite, Eclipse, Lotus Notes, Minecraft and OpenOffice. Java is known as the core foundation for developing Android applications, here are some of the unique features:

  • Application portability
  • Interpreted language
  • Extensive network library
There are a lot of reasons why Java might be a good first programming lanugage to learn:
  • Java has been around for a really long time so it's well established so it has a lot of great books and online resources to help further one's knowledge.
  • There are more Java programmers than any other type of programmer so a lot of people already know Java and it's easier to seek out help or mentorship from someone.
  • Java derives its syntax from C, so if you learn Java, learning more lanuages like Javascript, C#, or C++ will be much easier.
  • It's an object-oriented language, so a lot of fundamental coding concepts you learn inside of Java is transferable to other languages.

Why Java is Unique

Java, unlike JavaScript, is Statically typed as opposed to dynamically typed. This basically
mean's whenever we declare a variable we have to state what each variable will hold. Some
variable's will hold numbers, another will hold text, and another will hold something else.
Unlike in lanuages like Ruby or JavaScript, you won't know you have an error until you run
the code and doesn't work in some way.
