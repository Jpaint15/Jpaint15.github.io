C Language

C Lanuage is a structure-oriented, middle-level programming lanugage used mostly for lower-level applications.
Even though C is for more lower level applications, it's used by software developers, computer engineers, and anyone
in Information Technology, Engineering, Management, etc. Major organizations that are linked to C lanuage are
Microsoft, Apple, Oracle, Cisco, and Raytheon. It's also used in specialized industries such as:

  • Embedded Systems
  • Systems Programming
  • Artificial Intelligence
  • Industrial Automation
  • Computer Graphics
  • Space Research
  • Video Game Programming
  • Image Processing

Here are some attributes of C Programming

  • Procedural Language: Instructions in a C program are executed step by step
  • Portable: You can move C Programs from one platfrom to another
  • Speed: C Programming runs faster than most other programing languages
  • General Purpose: C Programming can be used to develop operating systems, embedded systems, databases, etc.
